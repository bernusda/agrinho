# Agrinho 2026 - Categoria Desenvolvimento Front-End 🌾💻

## 📝 Sobre o Projeto
Este projeto foi desenvolvido como parte do **Concurso Agrinho 2026**, em parceria com a plataforma **Alura**. O objetivo principal é unir a tecnologia do desenvolvimento web (Front-End) com a conscientização sobre a sustentabilidade e inovação no agronegócio.

A aplicação consiste em uma Landing Page interativa focada nos pilares do Agro Tecnológico e conta com um **Simulador de Economia Hídrica**, demonstrando como softwares e a tecnologia de precisão podem ajudar a poupar recursos naturais preciosos no campo.

---

## 🚀 Funcionalidades
* **Design Responsivo:** Interface que se adapta perfeitamente a computadores, tablets e celulares.
* **Seções Informativas:** Explicação visual e direta sobre o uso de IoT, IA e Gestão de Água no Agro.
* **Simulador Dinâmico:** Uma calculadora em JavaScript que simula o impacto positivo (em litros de água economizados) com base no tamanho da área cultivada (hectares).

---

## 🛠️ Tecnologias Utilizadas
Para a construção deste ecossistema web, foram utilizadas as tecnologias fundamentais do Front-End:
* **HTML5:** Estruturação semântica de todo o conteúdo da página.
* **CSS3:** Estilização moderna utilizando conceitos de *Flexbox*, *Grid Layout* e *Media Queries* para responsividade.
* **JavaScript (ES6):** Manipulação do DOM e lógica matemática para a interatividade do simulador.
